# DM Forge — Dungeon Master Toolkit

A full-featured D&D 5e (2024 PHB) DM management application.

## Features
- **Character Sheet** — Full 2024 PHB character builder with spells, equipment, PDF export
- **Campaign Tracker** — Multiple campaigns, session logs with XP, NPCs, loot, DM notes
- **Combat Tracker** — Initiative order, HP tracking, conditions, combat log, dice roller
- **Monster Compendium** — SRD monsters with full stat blocks, custom NPC creator, send to combat

## Deploy to GitHub Pages
1. Create a repo named `dm-forge`
2. Go to Settings → Pages → Source: GitHub Actions
3. Push this folder's contents to the `main` branch
4. App will be live at `https://Csand1303.github.io/dm-forge/`

## Local Development
```
npm install
npm run dev
```
