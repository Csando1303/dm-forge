# DM Forge — Dungeon Master Toolkit

A full-featured D&D 5e (2024 PHB) DM management app.

## Deploy to GitHub Pages

1. Create a **public** GitHub repo (any name)
2. Go to **Settings → Pages → Source** and set it to **"GitHub Actions"**
3. Push this folder's contents to the `main` branch
4. The app auto-builds and deploys — live at `https://Csand1303.github.io/dm-forge/`

> The `base` path is set automatically from the repo name — no config needed.

## Local Development
```bash
npm install
npm run dev
```
